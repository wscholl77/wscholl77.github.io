Project 1 Reflection

What issues or challenges did I face completing this project?

  1. When I ran the HTML validator for the image gallery, it didn’t like the section tags I used for the images. It also wouldn’t allow the fig caption tag to be nested within it.
  2. I wanted to add a small picture of myself within the header on the right side but couldn’t configure it correctly. It made the header way too big and I wasn’t sure how to configure a div class to fix it.
  3. The main challenge I believe I had with this assignment is making a unique and original webpage.
  4. I couldn’t easily create a dropdown menu to nest the gallery page within the project 1 menu.
  5. I had some difficulty understanding the question: “List four elements that comprise writing relevant content links”.

How did I resolve them?

  1. I changed the section tag to the figure tag per their recommendation.
  2. I left the picture out of the header.
  3. I couldn’t resolve this challenge but believe as I gain more experience it will improve.
  4. I setup the menu so gallery would appear only when project 1 or the gallery page was being viewed.
  5. I went through all the articles under Week 1 content several times and answered as best as I could.
