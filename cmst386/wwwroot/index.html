Project List - Spring 2021

Project 1: SEO Best Practices, Accessibility, User Engagement

Create multiple HTML5 pages using CSS3, SEO best practices, user engagement, accessibility, file paths, FTP, and a photo gallery.

Project 2: Website Proposal (Plan for Project 4)

Create a proposal for the Final Website Project that explains the topic, purpose, goals, target audience(s), description of the site and business, organization, testing, security and search engine optimization/marketing.

Project 3: Responsive Design and CSS Preprocessors

Students will create a web site that incorporates using CSS preprocessing with LESS or SASS, website analytics and conversion of the project to responsive design.

Project 4: Final Website

Students will design a real-world website of 10 pages based on the Project 2 proposal/plan. Students will implement JavaScript snippets, form validation, icons and search function. The goal is to develop a real world website based on a proposal/plan including that incorporates form development, a basic understanding of how to implement a search function, analyze their website to determine how best to categorize the navigation to meet best practices, and use other sources such as JavaScript snippets, icons and search function appropriately within the site.
